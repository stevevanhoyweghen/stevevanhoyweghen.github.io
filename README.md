Steve Van Hoyweghen his photo portfolio on GitHub.

Most content is generated with a generator I wrote. The exif and itpc data from the pictures are used to generate the site. I will share the implementation once all proves to be production ready.

Some background

I have established a complete workflow covering: Lightroom -> exif/itpc cleanup -> thumbnail generation -> page generation, on a Windows computer making use of Python, Jinja2, VIPS, and exiftool. Since all these tools are available on linux and Mac as well, porting the workflow to another OS will be easy.

More pictures will be added later. For the impatient: https://stevevanhoyweghen.wordpress.com/
